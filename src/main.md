## В данной папке располагаются основные скрипты, написанные для разработки бота

```
├── .env - Файл с переменными окружения (токен бота, настройки Redis, API и т.д.)
├── requirements.txt - Файл зависимостей Python со списком необходимых библиотек
├── Dockerfile - Докерфайл
├── docker.yml - Докер конфигурации 
└── snp_bot.py - Основной Python-скрипт телеграм-бота для анализа SNP
```
